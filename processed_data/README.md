## Processed (Cleaned) Data Directory

This directory contains all of the processed data. Here is a description of each of the files:

`census_data_cleaned.csv`: Census Data cleaned using `/scripts/process_census_data.py`

`postsecondary_schools_cleaned.csv`: Postsecondary School data cleaned using `/scripts/process_school_data.py`

`postsecondary_schools_cleaned.csv`: Pubilc School data cleaned using `/scripts/process_school_data.py`

See the [data codebook](../data_codebook.pdf) for the metadata for the three .csv files.
